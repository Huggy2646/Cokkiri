앞에 나오는 시연자: 이강우(새로운 계정, 어지민 친추 되있음)
자리에 있는 시영자: 김영수, 어지민

### 시연 시작

### 메인화면

강우 - 일일 퀘스트 버튼을 눌러 일일 퀘스트 목록을 보여준다.

### 스터디방

지민 - 메인 화면에서 스터디방 생성 후 어려운 문제 선택
영수 - 지민의 스터디방 입장
지민 - 친구 초대 버튼을 누르고 강우 초대
강우 - 우편함 보여주고 스터디방 입장
강우 - 문제 바꿔달라 채팅
지민 - 채팅에 답변 후 문제 바꾸기 (문제 이름 : A+B, A-B)

### 스터디방 시작

강우 - 문제 제출 (오답 : 예제만 통과하는 코드)
강우 - 힌트 보기 클릭
영수 - A+B 정답 코드 작성 후 공유 및 제출 버튼 클릭
지민 - A+B 정답 코드 작성 후 제출 버튼 클릭
강우 - 공유받은 코드를 복사하여 실행 및 제출 버튼 클릭
영수 나가고 강우 나가기

강우 - 유저 검색을 눌러 영수 계정으로 친추 보내기
영수 - 우편함에서 친구 요청 수락

### 정기 스터디

강우 - 스터디 모집 게시판 클릭
강우 - 이미 존재하며 모집 중인 정기 스터디 방 클릭 후 가입 신청

### 리뷰

강우 - 프로필에서 1000번(A+B) 버튼 클릭으로 리뷰 자동 완성 기능 실행
강우 - 문제 목록 1000번 문제 확인하기 클릭
지민, 영수 - 1000번 문제 리뷰에 댓글 달기

### 게임방 (종덕씨 2인분)

강우 - 메인 페이지에서 게임 방 생성 후 친구 초대(지민, 영수)
나머지 3인 - 방 입장
강우 - 레벨 2로 문제 설정

### 게임방 시작

강우 - 아이템 사용 (종덕 두번째 계정으로 사용)
나머지 사람들 - 아이템 사용 (강우 계정으로 사용, 아이템이 겹치지 않도록 사용)
강우 - 문제 정답 코드로 제출 (2개 다 완료)
강우 - 문제 제출 현황 확인
나머지 사람들 - 문제 정답 코드로 제출 (2개 다 완료)

### 게임 종료 및 순위 발표

강우 - 변경된 티어 보여주기
